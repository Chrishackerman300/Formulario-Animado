Proyecto Creado y Explicado por Christian Mendoza

En este proyecto realizare un formulario de datos de un usuario con HTML5, CSS3 y JS. Donde solo hare un 
maqueteado a la pagina y estilos con css, no hare validacion con javascript.

Utilizare javascript por unas cuantas lineas de codigo y es para que el usuario pueda ver su password.

El proyecto estara explicado y resumido dentro de los archivos y carpetas adecuadas para cada trabajo que realizare.
